弹出框showDialog 1.0

一、showDialog1.0更新内容：

  • 支持弹出框背景颜色、背景透明度可自定义
  • 支持弹出前、后CallBack操作
  • 支持弹出iframe高度自适应
  • 支持自定义消息弹出
  • 其他优化

二、使用说明

1、自定义弹出框颜色、背景透明度

参数：
showDialog.show({
id:"pic",//需要弹出的id，如果是弹出页面上的div，则该选项为必选
bgcolor:"#000",//弹出“遮罩”的颜色，格式为"#FF6600"，可选，默认为"#fff"
opacity:60//弹出“遮罩”的透明度，格式为｛10-100｝，可选
});
demo：
弹出一个id为“error”的box： demo
弹出一个颜色为橙色，id为“error”的box： demo
弹出一个透明度为60，id为“error”的box： demo
弹出一个颜色为橙色，透明度为60，id为“error”的box： demo

2、支持弹出前、后CallBack操作

参数：
showDialog.show({
id:"pic",//需要弹出的id，如果是弹出页面上的div，则该选项为必选
onPopupCallback:function(){alert("在弹出框弹出前运行这个函数")},//函数名，或者是匿名函数，可选
onCloseCallback:function(){alert("在弹出框关闭后运行这个函数")}//函数名，或者是匿名函数，可选
});
弹出前弹出“hello Word”： demo
弹出框关闭时弹出“hello Word”： demo
或者： demo

3、弹出iframe高度自适应

参数：
showDialog.show({
src:"ifram.html",//需要弹出的iframe的路径，必选
iWidth:512,//iframe的宽度，必选
iHeight:"auto"//如果要求iframe自适应，值为"auto"，如果不要自适应，则直接为需要设定的高度，如314，必选
});
自适应弹出： demo

4、自定义消息弹出

参数：
showDialog.show({
sMsg:"Hello...",//必选。内容可以为文本，也可以为html结构
sStyle:"width:500px;border:4px solid #ccc;padding:10px;background-color:#fff",//可选，自定义消息框的样式
sClass:"pop"//可选，自定义消息框的样式类名
sTime:3000//可选，多少秒后自动关闭，默认为点击弹出框以外的“遮罩”关闭
});
默认弹出一个消息，3秒后自动关闭： demo

5、其他

  • 在打开一个弹出框后，在弹出另一个弹出框时，不需要关闭前一个，如demo
  • 兼容旧版本，如弹出框showDialog.show('error')：demo，弹出iframe：showDialog.show('ifram.html','512','314','no')demo
  • 在iframe里面，如果父页面有showDialog脚本，则iframe不需要在引入该脚本，只需要：var showDialog=parent.showDialog; showDialog.show({/*......*/});
  • 参数过多，不方便写调用的话，可以这样：function p1(){showDialog.show({id:'error',bgcolor:'#f60',opacity:30,onPopupCallback:function(){alert('在弹出框弹出前运行这个函数')}});}然后直接使用 onclick="p1()"
